# Accessibility Documentation

## Overview

This document outlines the accessibility features and WCAG 2.1 AA compliance implementation for the Grill Business landing page. Our commitment to accessibility ensures that all users, regardless of their abilities or assistive technologies, can access and interact with our content effectively.

## WCAG 2.1 AA Compliance

### Compliance Level

The landing page is designed and tested to meet **WCAG 2.1 Level AA** standards, which include:

- **Level A**: Basic web accessibility features
- **Level AA**: Enhanced accessibility features for broader user coverage
- **WCAG 2.1**: Latest accessibility guidelines including mobile and low vision requirements

### Compliance Testing

Automated accessibility testing is performed using:

- **axe-core**: Industry-standard accessibility testing engine
- **@axe-core/playwright**: Integration with Playwright for automated testing
- **Manual testing**: Keyboard navigation, screen reader testing, and visual inspection
